## How to Contribute
- Read our [Use-Case Overview](docs/use-case-overview.md) to get up to speed.
- Check out open [Issues](https://github.com/cteague3/geekrush/issues) and [Project Board](https://github.com/cteague3/geekrush/projects).
- Submit ideas via GitHub Issues or Discussions.
- Fork the repo, make changes in a feature branch, and open a Pull Request.

## Tech Stack Overview
- **Backend:** .NET (C#), Azure Functions
- **Frontend:** Blazor WebAssembly (PWA-ready)
- **Optional Services:** Go microservices, Cosmos DB, Notification Hubs

## Code of Conduct
Be kind, inclusive, and constructive. GeekRush is built for fun and respect.

## Looking for Ideas?
- See our open Issues labeled `help wanted`
- Suggest new features or workflows
